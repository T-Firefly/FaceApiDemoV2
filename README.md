# FaceApiDemoV2 
工程主要包含FireflyApi和FaceApi两个功能模块：




## FireflyApi
FireflyApi包含了IDCard（身份证）、ICCard、二维码等相关功能，FireflyApi  根据自身的相关业务需求结合 SDK 的相关接口灵活的进行上层软件应用的开发。




## FaceApi
FaceApi主要是人脸识别应用编程接口，包含了人脸检测、人脸识别、人脸属性检测等相关功能。
